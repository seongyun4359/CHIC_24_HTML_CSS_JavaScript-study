#### 박스 테두리
: 박스에 테두리를 넣을 때는 두꼐, 형태, 색상에 해당하는 속성을 사용해야 한다.  
테두리 두께는 border-width 속성을 사용하고, 테두리 형태는 border-style 속성을 사용하며, 테두리 색상은 border-color 속성을 사용한다

- div 태그 : 블록 형식
- span 태그 : 인라인 형식
